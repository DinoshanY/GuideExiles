How to host with Github

First make a repository.

If your connected on github with ssh use ssh token if not use https token.

Go to where you have the code an open the termial, I am useing VSCodium and in the termial wright git init

Then git add .

Then git commit -m "text", Text is useally use to wright what has been change or added

Then git push token, The token is the one you get form the repository, Past it with Ctrl + shift + V

Now if you go to the repository you will see your code. Too host the code on github pages go to the Settings tab.

Then got to the Pages tab

On the pages tab you will see a drop down named none under source

Click on the none drop down and pick master

Now the site is hostet on github it will take som time for the site to be hostet so ekspekt an error

After some time the error wil be remove and you will see your website but som times youl will need to add /index.html or your main webiste html in the link to find your site
