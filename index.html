Class

There are 6 classes you can pick form the start and 1 unlockabel class before a boss battle in chapter 4, each class specialisel in diffrent or combind atributes:Strength,Dexterity andIntelligence. You can pick from Marauder:Strength, Ranger:Dexterity, Witch:Intelligence, Duelist:Strength &Dexterity, Shadow:Dexterity &Intelligence, Templar: Intelligence &Strength, and finally the unlockabel class Scion:All Atributes. each class has also three ascendancy exepte Scion she has one that is a mix of every ascendancy. Ascendancy is a special version of a class with uniqe passives. The classes have diffrent staring point in the passive tree.Strength classes have alot of melee passives nodes,Intelligence has alot of spell passives nodes andDexterity has alot of range and trap passives nodes. When picking a class it is importent to have fun so pick a class that will make you enjoy the game.
