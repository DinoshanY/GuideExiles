Commands

There are som usefull commands you can use in Path of Exiles and can help with gameplay.

Commands Catagory Function
/help Help Displays a list of most console commands.
/played Character Displays the length of time the current character has been played.
/age Character Displays how long ago the current character was created.
/passives Character Displays a summary of all passive skill points and the Deal with the Bandits reward.
/deaths Character Displays how many deaths the current character has incurred.
/remaining Area Displays how many monsters remain alive in the current area.
/hideout Town Sends you to your hideout. Only useable while in town.
/dance Character Perform a dance.
